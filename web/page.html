词库结构

块与块之间空一行；块首行是指令（正则触发词），其余行被触发后依次执行。多个指令同时命中时只执行最上面的。注释行以 //、##、&& 开头。

常用片段（点击插入）

%ID%%昵称%%群ID%%频道号%%子频道号%%参数1%%参数-1%%括号1%%AT0%%IMG0%%时间yyyy年MM月dd日 HH:mm:ss%%时间MMdd%%Time%%随机数1-100%

变量

局部变量：变:123（单字符名，仅当前语句块有效），使用 %变%。

全局变量：$全局写 名字 值$ / $全局读 名字 默认值$ / $全局删 名字$，或 %全局.名字%，持久保存、跨指令可用。$撤回$/$撤回 消息ID$ 撤回消息。

读写数据

$读 金币 %ID% 0$$写 金币 %ID% 100$$删除 路径$$读键列表 路径$

数据保存在插件 data/ 目录下，文件内容为「键 = 值」。推荐路径：%群ID%/金币 %ID%。

如果 / 分支 / 循环 / 返回

如果:%A%==1 如果尾 分支:%参数-1% 情况:早安 早 default: 嗯？ 分支尾 循环:i<=10 %i%\n i:[%i%+1] 结束 循环遍历:%J% x n 第%n%个：%x%\n 结束 跳出 继续 返回

条件支持 == != > < >= <=；[算式] 支持 + - * / % 和括号。

循环遍历:数据 项变量 序变量（单字符变量名，序变量可省）遍历 JSON 数组/对象（序=键）/逗号文本；循环内 跳出=break、继续=continue。

±md± 出现后整条消息按 Markdown 发送；±ark=类型|参数...± 支持 23/24/37 卡片；±引用± 本条回复引用触发者消息；±btn=文本;值|文本;值^下一行± 消息按钮（值为URL=链接、以/开头=填入输入框、其余=回调）；\n 或 \r 为换行。

排行榜 (JSON)

$排行榜写 排行榜.json 金币 %ID% 50$$排行榜读 排行榜.json 金币 参数 0$$排行榜读 排行榜.json 金币 值 0$

写入后自动按值降序排列；「参数」读取第 N 名的 ID，「值」读取第 N 名的数值（N 从 0 开始）。

数据库 (SQLite)

$数据库 声明 名称 文件.db$$数据库 执行SQL 名称 insert into 表 values ('a')$$数据库 查询SQL 名称 select * from 表$

返回 JSON：{"data":..., "errorMsg":"", "status":N}，status 为 -1 表示失败。声明可写在 #INITROBOT# 初始化块中。

调用 / 回调 / 内部指令

$调用 2000 指令名$$回调 指令名$[内部]指令名

调用：延迟 N 毫秒后触发指令并单独发送；回调：只能触发 [内部] 指令且输出合并到当前消息。
