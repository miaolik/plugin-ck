# plugin-ck 词库插件

ElainaBot_v2 框架下的 GQ 风格词库插件：用纯文本词库编写机器人问答与功能，兼容
[GQ 词库语法](https://docs.qq.com/aio/DYnRobmhxT0VHVUdY)，并附带 **Web 词库编辑器**（后台侧边栏「词库编辑器」页面）。

## 安装

将本仓库放入 ElainaBot_v2 的 `plugins/` 目录（目录名任意，如 `plugins/ck/`），重载插件即可。
词库文件位于插件目录 `dicts/*.txt`，运行数据位于 `data/`。

## 词库结构

- 语句块之间以**空行**分隔；块首行是指令（**正则触发词**，全字匹配），其余行被触发后依次执行。
- 多个指令同时命中时，只执行位于最上方的语句块（按文件名排序、文件内顺序）。
- 注释行以 `//`、`##`、`&&` 开头（需独立成行）。
- `#INITROBOT#` 块在词库加载时执行一次（适合放 `$数据库 声明$` 等）。
- `[内部]指令` / `#内部#指令`：不能被消息触发，只能被 `$调用$` / `$回调$` 触发。

```text
签到
如果:$读 签到 %ID% 0$==%时间MMdd%
今日已签到
返回
如果尾
签到成功！
$写 签到 %ID% %时间MMdd%$
```

## 变量

- 局部变量：`变:123`（第二个字符是 `:`，变量名一个字符，仅当前语句块有效），使用 `%变%`。
- 全局变量（持久化）：`$全局写 名字 值$`、`$全局读 名字 默认$`、`%全局.名字%`；也可在 Web 端管理。

## 控制语句

`如果:条件` / `如果尾`、`分支:值` / `情况:值` / `default:` / `分支尾`、`循环:条件` / `结束`、`返回`。
条件运算符 `== != > < >= <=`；`[算式]` 支持 `+ - * / %` 与括号（如 `[%金%+100]`）。

## 发送消息

文本直接写；`\n` / `\r` 为换行。多媒体与格式：

| 语句 | 说明 |
| --- | --- |
| `±img=URL或本地路径±` | 发送图片 |
| `±video=URL±` | 发送视频 |
| `±voice=URL±` | 发送语音 |
| `±file=路径±` | 发送文件 |
| `±at=用户ID±` | 艾特（`±at=0±` 为全体） |
| `±md±` | 本条消息按 Markdown 格式发送 |
| `±引用±` | 本条回复引用触发者的消息 |
| `±btn=文本;值\|文本;值^下一行±` | 消息按钮：值为 URL=链接、以 / 开头=填入输入框、其余=回调；`^` 分行，`\|` 分列 |
| `±ark=37\|提示\|标题\|副标题\|图片URL\|跳转URL±` | ARK 卡片（支持 23 / 24 / 37） |

## 函数（`$函数名 参数…$`，不可嵌套）

| 函数 | 说明 |
| --- | --- |
| `$读 路径 键 默认$` / `$写 路径 键 值$` / `$删除 路径$` | 读写配置数据（data/ 下「键 = 值」文件） |
| `$全局读 名 默认$` / `$全局写 名 值$` / `$全局删 名$` | 全局变量 |
| `$撤回$` / `$撤回 消息ID$` | 撤回触发消息 / 指定消息（机器人发的） |
| `$排行榜写 排行榜.json 键 参数 值$` | 写排行榜（自动按值降序） |
| `$排行榜读 排行榜.json 键 参数/值 索引$` | 读排行榜第 N 名的 ID / 数值 |
| `$数据库 声明 名 文件.db$` | 声明 SQLite 数据库（data/db/ 下） |
| `$数据库 执行SQL 名 SQL$` / `$数据库 查询SQL 名 SQL$` | 执行/查询 SQL，返回 `{"data":…,"errorMsg":"","status":N}` |
| `$访问 URL$` | GET 访问 URL 返回内容 |
| `$POST访问 URL 数据$` | POST 访问（数据为 `a=1&b=2` 表单或 `{"a":1}` JSON） |
| `$下载 本地路径 URL$` | 下载 URL 内容到 data/ 下 |
| `$URLEncoder 文本$` / `$URLDecoder 文本$` | URL 编解码 |
| `$字符串长 文本$` / `$字符包含 内容 关键词$` | 文本长度 / 包含判断 |
| `$替换 @ 原文@目标@替换$` / `$取中间 @ 原文@左@右$` | 替换 / 取中间 |
| `$分割 @ 文本@,$` / `$正则 @ 文本@(regex)$` | 分割成索引数组 / 正则匹配 |
| `$数组处理 add/del @ 数组@元素$` | 索引数组增删 |
| `$随机数 min max$` / `$随机字母 n 类型$` / `$随机汉字 n 0$` / `$随机英文数字 n$` | 随机 |
| `$是否为数字 文本$` / `$计算 表达式$` | 数字判断 / 算术计算 |
| `$调用 毫秒 指令$` | 延迟触发指令（输出单独发送） |
| `$回调 指令$` | 触发 `[内部]` 指令并合并输出 |

数组 / JSON 取值：`@[1,2,钝角] [2]`、`@{"a":1} [a]`、嵌套 `@%查% [data[0].表头1]`。

## Web 词库编辑器

后台侧边栏「词库编辑器」：

- 词库文件的新建 / 编辑 / 重命名 / 删除，保存即热重载（Ctrl+S）
- 沙盒测试：模拟消息 + 用户/群 ID，直接查看输出片段与错误
- 语法速查（点击片段插入编辑器）
- 变量使用说明：各类变量的说明与可一键复制的配置例子
- 全局变量管理、data/ 数据文件浏览与删除
- 设置：URL 访问/下载超时自定义（默认 300 秒）

接口位于 `/api/ext/ck/*`，全部走后台登录鉴权。

## 与 GQ 的差异

- 不支持 JS 脚本块（`#JAVASCRIPTSTART#` / `$执行JS$` / `$JS脚本$`）、画布函数、群管函数（禁言/踢人等由框架权限模型管理）。
- `$计算$` 为安全算术求值（非 JS）；`±at=±` 在 QQ 群开放平台下为文本假艾特。
- 循环上限 1000 次，调用深度上限 10 层；HTTP 访问/下载超时默认 300 秒（5 分钟），可在 Web 端「设置」中自定义（1-3600 秒）。
