# plugin-ck 词库插件

ElainaBot_v2 框架下的 GQ 风格词库插件：用纯文本词库编写机器人问答与功能，兼容
[GQ 词库语法](https://docs.qq.com/aio/DYnRobmhxT0VHVUdY)，并附带 **Web 词库编辑器**（后台侧边栏「词库编辑器」页面）。

## 安装

将本仓库放入 ElainaBot_v2 的 `plugins/` 目录（目录名任意，如 `plugins/ck/`），重载插件即可。
词库文件位于插件目录 `dicts/*.txt`，运行数据位于 `data/`。

内置示例词库 `dicts/变量示例.txt`：对机器人发 **示例菜单** 即可体验全部变量与功能（变量/时间/随机/如果/分支/循环/循环遍历/读写/全局/签到/数组/访问URL/多媒体/ARK/按钮/小按钮/引用/撤回/自动撤回/调用/主动消息/召回/成员查询/邀请链接/频道管理等），也是词库开发的参考模板。另内置 `dicts/修仙游戏.txt`：发 **修仙** 体验用词库写的完整小游戏（MD 排版 + 回调按钮 + 读写/排行榜综合应用）；`dicts/群管功能.txt`：群里发 **群管** 打开管理面板（违禁词管理与检测、警告记录、入群欢迎/退群提示，按群独立，设置仅群主/管理员）；`dicts/频道管理.txt`：频道里发 **频道管理** 打开面板（禁言/撤回/踢人/拉黑/身份组/帖子管理实战，管理类指令仅频道主/管理员）。

## 词库结构

- 语句块之间以**空行**分隔；块首行是指令（**正则触发词**，全字匹配），其余行被触发后依次执行。
- 多个指令同时命中时，只执行位于最上方的语句块（按文件名排序、文件内顺序）。
- 注释行以 `//`、`##`、`&&` 开头（需独立成行）。
- `#INITROBOT#` 块在词库加载时执行一次（适合放 `$数据库 声明$` 等）。
- `[内部]指令` / `#内部#指令`：不能被消息触发，只能被 `$调用$` / `$回调$` 触发。

```text
签到
如果:$读 签到 %ID% 0$==%时间MMdd%
今日已签到
返回
如果尾
签到成功！
$写 签到 %ID% %时间MMdd%$
```

## 变量

- 局部变量：`变:123`（第二个字符是 `:`，变量名一个字符，仅当前语句块有效），使用 `%变%`。
- 全局变量（持久化）：`$全局写 名字 值$`、`$全局读 名字 默认$`、`%全局.名字%`；也可在 Web 端管理。

## 控制语句

`如果:条件` / `如果尾`、`分支:值` / `情况:值` / `default:` / `分支尾`、`循环:条件` / `结束`、`返回`、`跳转:行号`。
条件运算符 `== != > < >= <=`；`[算式]` 支持 `+ - * / %` 与括号（如 `[%金%+100]`）。

`跳转:行号` 跳到本层第 N 行（1 起，指令行不计）继续执行，行号可用变量（如 `跳转:%n%`）；配合 `如果` 可实现条件跳转，向后跳可实现类 goto 循环（有执行步数上限保护）。

循环内可用 `跳出`（break）与 `继续`（continue）；`$回调$` 子块内的 `跳出` / `继续` 会穿透回调，作用于调用方所在的循环（例如回调里 `如果` 判断不通过时可直接跳出外层循环）。

`循环遍历:数据 项变量 序变量` / `结束`：遍历 JSON 数组、JSON 对象（序变量=键）或逗号分隔文本；
变量名均为单字符，序变量可省略。适合提取 `$访问$` 返回的 JSON 数组、或配合 `$读键列表$` 遍历本地配置：

```text
查列表
R:$访问 https://example.com/api$
循环遍历:@%R% [data] x n
第%n%个：%x%\n
结束
```

## 发送消息

文本直接写；`\n` / `\r` 为换行。多媒体与格式：

| 语句 | 说明 |
| --- | --- |
| `±img=URL或本地路径±` | 发送图片 |
| `±video=URL±` | 发送视频 |
| `±voice=URL±` | 发送语音 |
| `±file=路径±` | 发送文件 |
| `±at=用户ID±` | 艾特（`±at=0±` 为全体） |
| `±md±` | 本条消息按 Markdown 格式发送；块内所有文本片段（标题/多张 `$MD图片$`/`$MD代码$` 代码框/`$MD表格$` 表格/引用行等）合并成一条原生 MD 消息发出，`±btn=±` 按钮挂在同一条上 |
| `±文本±` | 强制纯文本（markdown 语法原样显示） |
| `±无后缀±` | 跳过全局 markdown 后缀 |
| `±自动撤回=秒±` | 本次发出的消息 N 秒后自动撤回 |
| `±引用±` / `±引用=引用消息ID±` | 本条回复引用触发者的消息 / 引用指定消息（值为 REFIDX，可用变量） |
| `±btn=文本;值\|文本;值^下一行±` | 消息按钮：值为 URL=链接、以 / 开头=填入输入框、以 > 开头=普通指令按钮（点击后以用户身份直接发送该指令）、其余=回调；`^` 分行，`\|` 分列 |
| `±小按钮=...±` | 同 `±btn±` 语法，按钮字号更小（font_size=small） |
| `±ark=37\|提示\|标题\|副标题\|图片URL\|跳转URL±` | ARK 卡片（支持 23 / 24 / 37） |

数组 / JSON 取值：`@[1,2,钝角] [2]`、`@{"a":1} [a]`、嵌套 `@%查% [data[0].表头1]`。

进退事件通知：在词库里建下列名字的词库块，对应事件发生时自动触发（块不存在则忽略）：`入群欢迎`（用户入群）/ `退群提示` / `机器人入群`（机器人被拉进群）/ `机器人退群` / `添加好友` / `删除好友` / `入频道欢迎`（用户加入频道）/ `退频道提示`。块内可用 `%ID%`（当事人）`%昵称%`（频道事件才有）`%群ID%` `%频道号%` `%操作人ID%`。文本输出的默认去向：群事件主动发到该群；频道事件发私信给该用户（想发到子频道就在块里用 `$主动频道 子频道ID 内容$`）；好友/机器人退群事件发私聊。需机器人订阅对应事件（频道进退为 GUILD_MEMBER_ADD/REMOVE）。

发帖审核：内置词库 `dicts/发帖审核.txt`（默认关闭，按频道分开设置，多频道互不影响）——在目标频道内发送「发帖审核」看菜单：「发帖审核开启」后，有人发帖命中违禁词（「添加发帖违禁词 词」维护）即自动 `$删帖$`，私信通知发帖用户，并向「发帖提示频道 子频道ID」设置的子频道发提示；设置类指令仅频道主/管理员可用。通知里的帖子标题会脱敏：命中的违禁词用 *** 遮掩；AI 审核判不合规时标题整体隐藏，避免机器人复述违规内容。另可「内容审核开启」接入 AI 内容审核（违禁词未命中时再审）：百度密钥没填时用内置接口，填了自动改走百度云（密钥全频道共用，Web 面板「设置」可填写并测试连通）。基于「发帖通知」事件块实现，需私域 FORUM 事件权限（公域事件不带内容，无法审核）。

帖子事件通知：在词库里建名为 `发帖通知` / `帖子评论通知` / `帖子回复通知` 的词库块，有人发帖/评论帖子/回复评论时自动触发（文本输出主动发到当前子频道）。块内可用 `%帖子ID%` `%评论ID%` `%回复ID%` `%帖子标题%` `%帖子内容%` `%发布时间%` `%ID%`（发布人）。需机器人订阅论坛事件；公域 `OPEN_FORUM_*` 事件官方不带内容，私域 `FORUM_*` 事件才有标题/内容/ID。评论列表与直接回复某条评论官方暂未开放接口，可在通知块里用 `$主动频道$` 发消息提醒。

## Web 词库编辑器

后台侧边栏「词库编辑器」：

- 词库文件的新建 / 编辑 / 重命名 / 删除，保存即热重载（Ctrl+S）
- 沙盒测试：模拟消息 + 用户/群 ID，直接查看输出片段与错误
- 语法速查（点击片段插入编辑器）
- 变量使用说明：各类变量的说明与可一键复制的配置例子
- 全局变量管理、data/ 数据文件浏览与删除
- 设置：URL 访问/下载超时自定义（默认 300 秒）
- 词库文件开关：文件列表每个词库可单独启用/禁用，禁用后不参与触发（仍可编辑）

接口位于 `/api/ext/ck/*`，全部走后台登录鉴权。

## 与 GQ 的差异

- 不支持 JS 脚本块（`#JAVASCRIPTSTART#` / `$执行JS$` / `$JS脚本$`）、画布函数、群管函数（禁言/踢人等由框架权限模型管理）。
- `$计算$` 为安全算术求值（非 JS）；`±at=±` 在 QQ 群开放平台下为文本假艾特。
- 循环上限 1000 次，调用深度上限 10 层；HTTP 访问/下载超时默认 300 秒（5 分钟），可在 Web 端「设置」中自定义（1-3600 秒）。
